# Password Strength Checker

This Python script evaluates the strength of user-entered passwords. It checks for:
- Minimum length (rejects passwords shorter than 8 characters)
- Use of lowercase, uppercase, digits, and special characters
- Common password weaknesses (like dictionary words)

It assigns a strength score and gives feedback to help users create stronger passwords.

## Features
- Score-based evaluation
- Real-time feedback
- Terminal-based interface

## Tech Stack
- Python 3
- Regex
